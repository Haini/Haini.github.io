This is the very first entry on this page and should demonstrate my ability to upload and change stuff.

\[a^2 + b^2 = c^2 \] \[\frac{n!}{k!(n-k)!}\]

And while we are still in awe of these beautiful equations I might mention a thing or two on bringing them to this blog in its today’s appearance.

First of all I followed this nice blog - and everything did work fine on localhost.

I copy pastaed following snippet into my C site.hs:

pandocMathCompiler =
    let mathExtensions = [Ext_tex_math_dollars, Ext_tex_math_double_backslash,
                          Ext_latex_macros]
        defaultExtensions = writerExtensions defaultHakyllWriterOptions
        newExtensions = foldr S.insert defaultExtensions mathExtensions
        writerOptions = defaultHakyllWriterOptions {
                          writerExtensions = newExtensions,
                          writerHTMLMathMethod = MathJax ""
                        }
    in pandocCompilerWith defaultHakyllReaderOptions writerOptions

And the following snippet needs to put everywhere you want to use MathJax:

<script type="text/javascript" async
  src="https://cdn.mathjax.org/mathjax/latest/MathJax.js?config=TeX-MML-AM_CHTML">
</script>

So if you are lazy (as me) just put it in your regular post template. One could implement a marker and add a context that will check the marker and insert the script only if necessary.

I did then proceed to add the missing dependencies pandoc and containers in my .cabal file so that the compiler would be happy. Working flawless on localhost I continued to push my first steps in github.io where it failed without much hesitation.

The reason was interesting though. Google Chrome seems to block scripts from unsecure context - which my mathjax link was at fist indeed as I used the http:// version from the blog mentioned above. Replacing it with the more recent version in my post everything started to work perfectly. Still took me half an hour to realize the cause as Google doesn’t seem to promote this We block somehting very obvious.
